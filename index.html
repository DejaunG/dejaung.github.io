FutureForge DB

Database React + Mantine

FutureForge DB is a cutting-edge project database developed using React and the Mantine library. This dynamic web application showcases my skills in modern web development, UI/UX design, and system administration. Hosted on a Linux Pi server using Apache and leveraging the open-source DuckDNS for domain management, this project demonstrates my ability to create full-stack solutions and deploy them in real-world environments. The database offers an intuitive interface for managing and showcasing various projects, highlighting my commitment to creating practical, scalable, and user-friendly applications.

AdversaGuard

Program Python

AdversaGuard is an advanced tool designed for the analysis and defense against adversarial attacks on image classification systems. It implements state-of-the-art techniques like Fast Gradient Sign Method (FGSM) for generating adversarial examples, coupled with sophisticated detection and reversion mechanisms. This project showcases the practical application of AI security concepts in protecting machine learning models from malicious inputs.

LLM Security

Research Paper

A comprehensive study on the vulnerabilities of Large Language Models like ChatGPT and DALL-E 3 to adversarial prompts. Explores attack vectors and proposes defense mechanisms for AI systems. This research delves into the ethical implications of AI vulnerabilities and offers innovative strategies to enhance the security and reliability of language models, contributing to the broader discourse on responsible AI development and deployment.

Multimodal Defense

Research Paper Poster Python

Research on detecting and mitigating multimodal adversarial attacks on Large Language Models. Investigates text and image input interactions, proposing innovative defense techniques. This project addresses the complex challenges of securing AI systems that process multiple types of data, showcasing interdisciplinary skills in computer vision, natural language processing, and cybersecurity.

AI Resilience

Research Paper

Introduces the Response Quality Score (RQS), a new metric for assessing AI model resilience against adversarial prompts. Provides insights into the robustness of leading AI models. This groundbreaking research not only advances the field of AI evaluation but also offers practical tools for developers and researchers to enhance the reliability and security of AI systems in various applications.
